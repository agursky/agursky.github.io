Making a New File in Google Docs

Making a new file in Google Docs is easy. Just click on the red "Create" button, and choose what kind of file you want to make. You can make a Document (for writing), a Spreadsheet, a Slideshow, and more. Once you have a file open, you can see click on the top to give that File a name.
