Intro to Google Docs

Google Docs is part of Google Drive. Most Cloud Computing websites like Google Drive let you save your files in them, but what makes Google Drive special is that it lets you make files in it too. That is what Google Docs is. It is used for making different kinds of writing files. It's a lot like Microsoft Office, which we have already used a lot.

    In Microsoft Office we use:
  • Microsoft Word to write on a page
  • Microsoft Excel to make Spreadsheets
  • Miscrosoft Powerpoint to make Slideshows

We can use Google Docs to make Writing Files, Spreadsheets, and Slideshows too. But when we do it on Google Docs it is saved in the Cloud, which means that if I start working on it on one computer, I can finish working on it later on another computer. Here are some other ways they are different.

Google Docs Microsoft Office
You can get your files from any computer You can only get your files on the computer you saved it on.
Google Docs saves what you write everytime you write a letter You have to save your work. If you forget, you lose whatever your wrote.
