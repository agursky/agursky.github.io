The Eye Dropper Tool

Today we are going to practice using the Eye Dropper Tool. The Eye Dropper Tool helps us fix mistakes by matching colors.

Sometimes when we are drawing on a page, we make a mistake. The eraser tool doesn't always help us, because the eraser tool ALWAYS draws white. Sometimes we don't want white, we want the color that we had when we started.

To fix this problem, we can match the color using the Eye Dropper Tool. This tool will match whatever color we had before, and we can draw over our mistake to make our drawing look the way it did.
