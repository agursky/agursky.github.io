Teaching Point:

Today we will learn how to change the color of writing in Kid Pix.

Changing the Text Color

One really cool thing about writing on the computer, is that we can write in any color we want. We can choose the color we want to type in before we type, or we can choose to change the color of our letters after we are already done.

To Change the color of our letters we choose the text that we want to change by highlighting those letters. Then, with the letters highlighted we pick a color. We will see our letters change as soon as we click on the color.

Today's Assignment

Change the color words on the screen to the color that they should be by highlighting them.

Extension Activity

Write some of your own color words and change them to the right color.
