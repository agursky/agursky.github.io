Teaching Point:

Today we will learn how to use the "T Tool" to type in Kid Pix

T Stands for Text

When we want to write on the computer, we use the Keyboard. The words that we type on the screen are called “Text”. In Kid Pix when we want to write Text, we use the “T Tool”.

First we click on the “T Tool”, and then we click on the part of the screen that we want to write on. This makes a “Text Box”. We can make the Text Box different sizes if we want. Anything that we type will go into the Text Box.
