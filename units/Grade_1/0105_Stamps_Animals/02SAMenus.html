Stamp Menus

There are a lot of different stamps to choose from, and sometimes it can be hard to find the one we want. Most of the Kid Pix Stamps are in different folders or menus on the computer. We can try to find a certain kind of stamp by looking for a folder by name. To switch between different folders:

  • Click on the Stamp Menu Button and find the menu you want from the list OR
  • Switch between lists using the up and down arrows next to the menu button.

Today's Assignment

You will make a favorite stamp menu page. You must fill each box with only stamps from one folder. You can choose which folder to use to fill up each box with.
