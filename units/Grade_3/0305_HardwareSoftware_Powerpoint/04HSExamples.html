Teaching Point

Students will create the outline for the slides in their Hardware Software Slideshow

Examples of Hardware and Software

The rest of your slideshow will be examples of Hardware and Software. You can start by picking one example of Hardware you can think of, and making a new slide to talk about it. Add one new slide. The title should be the piece of Hardware you want to talk about. In the Text Box, write three sentences about that Hardware. You can say what it does, what it looks like, how it helps us, or anything else that comes to mind.

When you are done you should make two more Hardware Example Slides, and then 3 Software Example Slides. Remember, each slide only talks about ONE example of Hardware or Software.

When you are finished with all of your slides your slideshow should have a total of 8 slides:

  • Slide 1: Title Slide
  • Slide 2: Introduction
  • Slide 3: 1st Hardware Example
  • Slide 4: 2nd Hardware Example
  • Slide 5: 3rd Hardware Example
  • Slide 6: 1st Software Example
  • Slide 7: 2nd Software Example
  • Slide 8: 3rd Software Example

Today's Assignment

Create the rest of your slides, but don't write in them. Only write the titles. Use the list above as a guide.
