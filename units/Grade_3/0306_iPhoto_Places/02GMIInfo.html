Teaching Point:

Today we will learn how to add information to pictures in iPhoto.

Photo Info

iPhoto is not only good for working with large groups of pictures. It can also be used to change or add to one picture. We can add information about a picture using the info button. Each picture has it's own set of information, such as when it was taken, where it was taken, who was in it, and what it's name is. To add information to your picture:

  • Click the picture you want to change.
  • Click Info on the bottom right. This will open up an info window.
  • You can change the name at the top if you want.
  • Click "Add a Description" to write what the picture shows.
  • Click "Assign a place" to write where the picture was taken.

Today's Assignment

Add information to your pictures. You must click on Assign a Place and write where the picture was taken. If you know the city, write it. If you only know the country, then just write that. You don't have to add other information, but every picture must have a place.
