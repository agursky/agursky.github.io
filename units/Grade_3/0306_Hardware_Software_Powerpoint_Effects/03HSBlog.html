Why Do We Export?

When we are done with our work, we want to Export it and put it on our blog. The reason we Export is because a Powerpoint file will not go on our Blog. Our Blogs only know how to have web files. So Exporting makes a copy of your work for the web. Once you have that web file, you are able to put it on your blog. When you export name your work whatever the name was, with an X at the end of it. To Export in Powerpoint:

  • Go to FILE
  • Click SAVE AS MOVIE
  • Make sure to save it on your DESKTOP
  • You can add an X to the end of it's name, so that you can tell it apart from your non export powerpoint file.
