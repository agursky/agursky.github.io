Helpful Properties in CSS

We can change the properties on ANY HTML tag. Today we are going to use properties to make our page neat. One important thing to know is that for all of our properties that have to do with size - web pages measure size in something called pixels which are written as "px" like "10px" means 10 pixels.

Here are some properties to play with:

  • For the div tag
    • Use width to control the size of how much of the page the div tag takes up. Use between 0 and 100%.
    • Use margin to set how far from the edge of the page it is. Set the margin to auto. This will make it stay in the center of the page the whole time.
  • for the img tag
    • use width to control the width of the images. This is useful, because it will make all of the images the same width, which will neaten up your page.
    • Use margin to control how much space is in between each image. Anything between 1 and 20px is fine.
