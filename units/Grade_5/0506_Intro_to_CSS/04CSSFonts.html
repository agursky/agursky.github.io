Using Fonts in CSS

In computers, fonts are the shapes of the letters on the page. When using fonts on a website, the font not only has to be on your computer, but also be ont the computer of the person viewing your website. There are only a few fonts that can be used in CSS without extra help.

To set the font in CSS

  • in the html tag set the font-family property to the font you want.

How to get the Really Good Fonts

To get the really cool fonts, you can use many sites. A very popular one is Google Fonts. To use a Google Font:

  • Find the Font you want to use on Google Fonts and click on the quick use button.
  • Copy the special code they give you and paste it to the head of your html page.
  • in your CSS use the font-family property, and writ the name of the font in quotation marks.
