Project Intro

For our next project we are going to learn about a new programming language called CSS. CSS is a very simple language that is used to change the way that an HTML Page looks. Colors, Sizes, and Layout (where things are on the page) are all set using CSS.

Teaching Point:

Today we will learn what CSS looks like, and how to put it into our HTML Page.

Using Style tags for CSS

There is more than one way to put CSS into a page. The way that we are going to do it, is by using style tags. Style tags are HTML tags. Unlike the HTML tags we have used so far, these tags go into the HEAD of the HTML page. Remember, we said that in an HTML Page, the body had all of the writing and pictures for the page, but the head had all of the information ABOUT the web page. The style information about the webpage goes in between the two HEAD tags. See the example below.

All of our CSS goes in between the two style tags. The way to write CSS is to use:

  • Selector - the HTML tag you want to change
  • Property - the thing about the HTML tag that you want to change
  • Value - the thing you want to change it to.

Here is an example that changes the color of the paragraph text to red, and changes its size to 14px. As you can see the way CSS works is pretty simple. You write:

  • The tag you want to change (like p or h1)
  • Open Curly Brackets
  • The thing you want to change about that tag.
  • A colon (:)
  • The thing you want to change it to.
  • Close Curly Brackets.
