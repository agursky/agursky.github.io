Project Intro

For our next project we are going to continue using Scratch, but we are going to start something new. We are going to use Scratch to tell a joke. This will be a simple project that will be practice for a future project where we tell a whole story. We will use the skills that we have been learning this year so far to make this happen, and we will learn some new skills to make us better programmers.

Teaching Point:

Today we will see how we can use Scratch to show characters having a conversation.

Using Scratch to Tell a Story

Using Scratch for stories doesn not involve any new skill. It is actually easier than a lot of the stuff that we've been doing. We can tell the character what to do, how long to do it for, and what order to do it in. We put everything we want, in order, under the Green Flag.
