Using Comments in HTML

A big webpage can have a lot of code on it, and if we or someone else wants to look at it, it can be very confusing. That is why we use comments. Comments can be helpful to us when we are going over our HTML code and trying to see where something is, or remember why we wrote it. Unlike tags, when we write comments into our HTML, the web browser ignores it. To add a comment to your code just write a less than sign, an exclamation point, and two dashes. Anything you write after that is your comment. To end your comments write 2 more dashes and a greater than sign. See the example below.
