Project Intro

For our next project we will be learning how to write HTML. HTML is a coding language that is used to make websites. Every webpage that you go to uses HTML to make writing, pictures, videos, and anything else that goes on the page. The page that you are reading right now was made using HTML. We will be using a free app called Brackets to write our code. We will write HTML code, and see our webpages using web browsers like Safari, Google Chrome, or Firefox.

HTML Tags

The way that HTML works is simple. The web browser (like Safari, Chrome, etc) reads HTML codes called tags to know what you want your website to say. If I wanted to write a paragraph, I would write whatever I wanted in between two paragraph tags. If I wanted to write a title, I would write the title in between two title tags. An HTML tag is a letter or letters in between a less than sign (<) and a greater than sign ( >). Here is an example of a pair of paragraph tags: <p>I write my paragraph here.</p>. Notice how the ending paragraph tag has a forward slash before the p. Almost all HTML tags work like this - an opening tag, and a closing tag which is exactly the same as the opening tag, except that it has a forward slash in the beginning.
