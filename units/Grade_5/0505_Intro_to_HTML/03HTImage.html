Adding Pictures to our Webpage

As we learned before, to add something to our webpage we just need to find out what the tag is. For pictures the tag is img. But picture tags work differently than regular tags in two important ways.

  • There is NO closing tag.
  • The img tag has special properties called ATTRIBUTES that tell about the img tag. We need to tell it where to get the image from.

How do we Write Attributes?

Attributes go after the tag, but BEFORE the greater than sign. You write:

  • The attribute name
  • and equals sign
  • The attribute value in quotation marks.
