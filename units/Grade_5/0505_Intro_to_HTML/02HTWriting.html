Teaching Point:

Today we will learn how to write text in HTML.

Writing onto Webpages.

Working with HTML is very easy. All that you need to put something on your page is to find out what the tag for that thing is. There are two types of tags that we use to write text in HTML - Title tags and Paragraph tags.

  • Paragraph tags are used to write regular text. Write what you want in between two "p" tags, like in the example below.
  • Title tags are for different titles. There are six different title tags. They all work the same, they are just different sizes. The tags are h1, h2, h3, h4, h5, and h6. h1 is the biggest. h6 is the smallest.

Use at least one title tag, and at least one paragraph tag to write onto a webpage. It doesn't matter what you write. Today is about practice, so feel free to write something silly if you want.
