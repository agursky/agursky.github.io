CSS Classes

So far we have learned to set rules for HTML Elements using CSS. We can set a rule for P TAGS that will change all of the paragraphs on the page. We can set a rule for IMG TAGS that will change all of the pictures on the page. CSS Classes are for when you don't want to change ALL of the things on your page of a certain type (like ALL images or ALL paragraphs). Classes can be used to change only a few of those things on your page, or only one if that's what you want.

If my page has 4 paragraphs, and I want 3 of them to have green writing and 1 to have red writing, then that would be a good time to use a class. I would set a rule for P Tags in CSS to have green writing, but set a special rule for a CSS Class to have red writing, and write that the paragraph I want is part of that class. To do this:

  • Pick a class name. It can be anything you want.
  • In your HTML opening tag write class="className"
  • In your CSS write the class name with a dot in front of it (like this: .classname) and then set your rules in your brackets.
