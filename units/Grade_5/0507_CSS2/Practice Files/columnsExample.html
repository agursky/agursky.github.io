Sometimes we don't want everything to be stacked on vertically on our HTML Page. If we want two or more things next to each other, we have to write in special code to let the web page know to do that. To make this work, we have to use something called "Float". Float is a CSS rule that can be set to either left or right. It is called Float, because whatever is set to float sticks to the side of the page, and everything else on the page floats around it.
