Teaching Point:

Today we will learn how to choose different Code Blocks and put them together in Scratch.

Scratch Menus

In Scratch there are different menus for different kinds of codes that we can use. What are some of the menu's you see? We can use any of these codes to give the computer instructions. In Scratch each part of our app has it's own instructions. So if we have a game with a cat and a mouse, there will be once set of codes for the cat, and one set of codes for the mouse. When we find a code we want, we just drag it into the workspace.

Sometimes codes work together. When we want codes to work together we attach them. The code on top goes first and the code on the bottom goes next. We can attach as many codes as we want. When we want to see the codes work, we can click on them.
