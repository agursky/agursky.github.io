Teaching Point:

Today we will learn how to add new characters to our Scratch File.

Adding Characters

There are many different characters you can add in Scratch. We can "Import" them by clicking on the button near where our characters are. Most of the characters you can import will already have more than one costume to use. REMEMBER - each character has it's own set of codes, so make sure you double click on the new character if you want your codes to go to them and not to the cat.

Today's Assignment

Add a new character and give them a dance to do while with the cat.

Extension Activity

Add button controlled color effects to the new character. Add as many characters as you want and make a dance party.
